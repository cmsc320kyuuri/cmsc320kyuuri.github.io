Data Collection and Parsing¶

For this tutorial, we pulled data from this dataset on Kaggle on the top 1000 Twitch streamers (based on watch time) from a specific year. This dataset includes information on each of the listed streamers such as number of followers, followers gained over the past year, and total watch time in minutes. We downloaded the .csv file containing the dataset and uploaded it to our Jupyter folder. Then, using the pandas software library for Python, the data within the csv file is put into a dataframe using the function "read_csv". This will allow us to perform various functions that will help us analyze the data. Thankfully, the data is already well organized in a .csv file, so there is no need to do any extra collection or parsing.

Hypothesis Testing¶

Null Hypothesis assumes that there is no association between two variables\ explain what null hypothesis is\

Streaming Times vs Followers Gained¶

The Null Hypothesis: There is no relationship between the streaming times vs followers gained.\ Let's use seaborn's regplot to do this

In [19]:
plt.figure(figsize=(10,10))
plt.rc('xtick', labelsize=20)
plt.rc('ytick', labelsize=20)
plt.rc('axes', labelsize=25)
sns.regplot(x = "stream_time", y = "followers_gained", data = streaming_followers_gained_df).set(title="Time Spent Streaming vs. Followers Gained Over the Course of a Year")
plt.show()

Seems like there is not much correlation between stream time and followers gained. There are many outliers... lets calculate the p-value. if the p-value is large (greater than 0.05) we can safely accept the null hypothesis
